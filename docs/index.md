# LiftLog documentation

LiftLog is a private, browser-local workout programming app. This documentation covers everyday planning workflows and the local development setup.

## For using LiftLog

- [Using LiftLog](using-liftlog.md) explains the core concepts, a first-program workflow, workout generation, and summaries.
- [Data management and backups](data-management.md) explains exercise-library imports, SQLite backups, restores, and folder backups.
- [Mesocycle spreadsheets](mesocycle-spreadsheets.md) explains the Excel export/import workflow and its replacement behavior.

## For working on the project

- [Development guide](development.md) covers setup, commands, architecture, tests, and GitHub Pages deployments.

## Start safely

LiftLog stores data in the current browser on the current device. Create a complete program backup before making a destructive change, importing a spreadsheet, clearing browser data, or moving to another device. See [data management and backups](data-management.md).
